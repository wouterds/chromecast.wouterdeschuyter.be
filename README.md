# React Web Boilerplate

Personal taste of a fresh React web project.

## Setup

```shell
npm install
```

### VSCode

```json
{
  "flow.useNPMPackagedFlow": true,
  "javascript.validate.enable": false,
  "eslint.autoFixOnSave": true,
}
```

## Development

```shell
npm run hot
```

## Building

```shell
make build
```
